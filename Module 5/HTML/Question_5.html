Key Features of HTML5

Explore the powerful capabilities introduced in HTML5.

1. Semantic Elements

HTML5 provides new semantic tags like <header>, <footer>, <section>, and <article> that make the structure of web pages more meaningful and accessible.

2. Multimedia Support

With HTML5, you can embed audio and video directly using the <audio> and <video> tags.

The <canvas> element allows drawing graphics and animations using JavaScript, but here we simply demonstrate the element's inclusion.
