1. html: The root element that defines the entire HTML document.
  2. head: Contains meta-information about the document such as title, character set, and linked resources.
  3. title: Sets the title of the web page, which appears in the browser tab.
  4. meta: Provides metadata like keywords, author, and viewport settings. Usually placed inside the head tag.
  5. link: Used to link external resources like CSS files to the HTML document.
  6. style: Embeds CSS styles within the HTML document.
  7. script:Embeds or references JavaScript code for interactive behaviors.
  8. body: Encloses the visible content of the HTML document like text, images, and other elements.
  9. h1 to h6: Heading tags used to define headings, with <h1> being the most important and h6 the least.
  10. p: Defines a paragraph of text.
  11. a: Creates hyperlinks to other web pages, email addresses, or files.
  12. img: Embeds an image into the page. Requires a 'src' attribute.
  13. ul: Creates an unordered (bulleted) list.
  14. ol: Creates an ordered (numbered) list.
  15. li: Represents a list item inside <ul> or <ol>.
  16. div: A generic container for grouping and styling elements using CSS.
  17. span: An inline container used for styling a portion of text or grouping inline-elements.
  18. table: Defines a table structure to display tabular data.
  19. tr: Table row element, used within table.
  20. td : Table data cell element, used within tr .
