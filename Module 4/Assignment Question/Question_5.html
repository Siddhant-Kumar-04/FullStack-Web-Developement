HTML
Defines the structure and content of web pages using markup elements.
CSS
Used to style and layout web pages, including colors, fonts, and responsive design.
JavaScript
Enables dynamic behavior on websites, such as interactive forms, animations, and user input handling.
React
A JavaScript library for building fast and interactive user interfaces using a component-based approach.
Node.js
Enables running JavaScript on the server side to build scalable backend applications.
Express.js
A lightweight Node.js framework used to build RESTful APIs and web applications.
MongoDB
A NoSQL database that stores data in flexible, JSON-like documents, ideal for scalable applications.
MySQL
A relational database management system used to store structured data and perform SQL queries.
Git
A version control system that tracks changes in source code during software development.
GitHub
A web-based platform for hosting and collaborating on code repositories using Git.
Docker
A tool that packages applications and their dependencies into containers for consistent environments.
Webpack
A module bundler that compiles JavaScript modules and assets into a single optimized output.
REST API
An architectural style for designing networked applications using HTTP requests for communication.
JWT (JSON Web Tokens)
Used for securely transmitting information between client and server as a JSON object, often for authentication.
Postman
A tool for testing and documenting APIs by sending HTTP requests and viewing responses.
